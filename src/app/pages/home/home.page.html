Welcome to the Legion Directory

This application is a wrapper for the 501st Legion Member API v3. There is no documentation for the API at the moment, so everything here was pieced together from forum posts.

Currently, you can view the following:

  • A list of all detachments.
  • A specific detachment, including officers.
  • A list of all garrisons and outposts.
  • A specific garrison or outpost, its officers, and its members.
  • A specific member, including all of the members registered costumes.
  • A verification page that looks up the given email address and returns the associated Legion ID if the member is active

Next on the development agenda:

  • Global search by Legion ID, email address, detachment name, garrison name, or outpost name.
  • Updated styling.
  • About, privacy, and terms pages.
